# ⛏️ Minecraft Server Manager - Mobile App

Aplikasi mobile untuk mengelola server Minecraft dengan tampilan modern dan fitur lengkap.

## 📱 Download

- **HTML App**: Download `index.html` dan buka di browser
- **Android Project**: Download `Android-Project.zip` untuk build APK
- **Full Package**: Download `MC-Server-Manager-Mobile.zip`

## ✨ Fitur

- 120 Java Edition versions (1.0 - 1.26)
- 106 Bedrock Edition versions
- 22 server templates
- 47+ plugins
- Cross-play support
- Real-time console
