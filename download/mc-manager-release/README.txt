CARA PENGGUNAAN:

OPSI 1: PWA (Progressive Web App) - RECOMMENDED
================================================
1. Buka file "index.html" di browser Chrome/Edge di Android
2. Tap menu (3 titik) → "Add to Home Screen"
3. App akan terinstall seperti APK biasa

OPSI 2: Build APK Sendiri
=========================
1. Install Android Studio
2. Buat project baru dengan "WebView" template
3. Copy index.html ke folder app/src/main/assets/
4. Update URL backend di file index.html
5. Build → Build Bundle(s) / APK(s) → Build APK(s)

OPSI 3: Online APK Builder
===========================
Gunakan layanan seperti:
- https://andideploy.com/
- https://appgeyser.com/
Upload file HTML dan build APK secara online
